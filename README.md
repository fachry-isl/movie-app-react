# TMDB Movie App with ReactJS

## Feature:

- Search Movie: Search movie by calling TMDB API
- Favorite : Favorite list of movies using PostgreSQL

## Stacks

- Frontend: ReactJS
- Backend: FastAPI and PostgreSQL
